HTML na prática

Sobre o Desafio

Este é um desafio prático de HTML, onde aplicamos conceitos básicos de estruturação de páginas web, ensinados até o momento.

Objetivos

  • Aprender a criar uma estrutura básica de HTML
  • Utilizar tags semânticas corretamente
  • Incluir imagens e listas
  • Implementar tabelas e formulários

Aprender a criar uma estrutura básica de HTML

Para criar uma estrutura básica de HTML, devemos utilizar as tags essenciais como <html>, <head> e <body>. Essas tags formam a espinha dorsal de qualquer página web e são fundamentais para a organização do conteúdo.

Exemplo de estrutura básica:

  • <!DOCTYPE html>: Informa ao navegador que o documento é HTML5.
  • <html lang="pt">: Começa o HTML e define o idioma como português.
  • <head>: Contém informações da página (como o título).
  • <meta charset="UTF-8">: Garante que os acentos e caracteres especiais apareçam corretamente.
  • <title>: Nome da aba no navegador.
  • <body>: Onde fica o conteúdo visível da página (textos, imagens, etc.).

Utilizar tags semânticas corretamente

As tags semânticas são importantes para a acessibilidade e SEO. Devemos utilizar tags como <header>, <footer>, <article> e <section> para estruturar nosso conteúdo de forma significativa. O porquê de usar essas tags é que elas ajudam os motores de busca a entenderem melhor o conteúdo da página, além de facilitar a navegação para pessoas com deficiência.

  • <header>: Cabeçalho da página, geralmente contém o título e a navegação.
  • <footer>: Rodapé da página, onde ficam informações de contato e direitos autorais.
  • <article>: Representa um conteúdo independente, como um post de blog.
  • <section>: Agrupa conteúdos relacionados dentro de uma página.

Incluir imagens e listas

Para incluir imagens, usamos a tag <img> com o atributo src para especificar o caminho da imagem. As listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>), e cada item da lista é definido com a tag <li>.

  • <img src="caminho/para/imagem.jpg" alt="Descrição da imagem">: Insere uma imagem na página.
  • <ul>: Cria uma lista não ordenada.
  • <ol>: Cria uma lista ordenada.
  • <li>: Define um item de lista.

Implementar tabelas e formulários

As tabelas são criadas com a tag <table>, e cada linha é definida com <tr>. As células são criadas com <td> para dados e <th> para cabeçalhos. Os formulários são criados com a tag <form>, e os campos de entrada são definidos com tags como <input>, <textarea>, e <button>.

  • <table>: Cria uma tabela.
  • <tr>: Define uma linha na tabela.
  • <td>: Define uma célula de dados na tabela.
  • <th>: Define um cabeçalho de tabela.
  • <form>: Cria um formulário para entrada de dados.
  • <input>: Campo de entrada de dados (texto, senha, etc.).
  • <textarea>: Área de texto para entradas maiores.
  • <button>: Botão para enviar o formulário ou executar ações.
